모듈러(modular) 연산

모듈러 연산이란 숫자 A를 B로 나눈 나머지를 구하는 연산이다.
A mod B = C로 표현할 수 있다.

모듈러 연산의 특징

모듈러 연산은 다음과 같은 특징을 갖는다.
1. (A+B) mod C = (A mod C + B mod C) mod C
2. (A-B) mod C = (A mod C - B mod C) mod C
3. (A*B) mod C = ((A mod C) * (B mod C)) mod C
